## Projeto Integrador SENAC-PA
Projeto integrador para o curso de desenvolvedor de sistemas do senac-pa

O projeto consiste no desenvolvimento de um sistema de compra de ingressos de cinema com seleção dos filmes,horários,assentos,tipo de sessão. Assim como opções para funcionários e gerentes cadastrarem filmes, alterarem informações dos filmes e gerar relatórios de gestão

### Instalação
1.Clonar o repositório do github e entrar no repositório
```bash
git clone https://github.com/Frico-eng/projeto-integrador.git
cd projeto-integrador
```
2.Instalar as dependências
```bash
pip install -r requirements.txt
```
3.Rodar a aplicação
```bash
python main.py
```
